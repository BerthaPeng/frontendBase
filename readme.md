该项目用作前端项目开发的模板
### js/lib        常用的库文件和插件或公用函数
### js/plugins    封装好的组件
### js/           js文件
### pages/        模板源文件（html）
### styles/       样式文件

** 所有模版源文件都位于pages目录下，前后端共用，
   需通过前端自动化工具的编译，才能生成到指定的static目录，供后台使用 **



###开发命令
- $ gulp         : 前端本地开发的命令
- $ gulp deploy  : 前端发布的命令


###在开发之前，npm install 安装依赖包
